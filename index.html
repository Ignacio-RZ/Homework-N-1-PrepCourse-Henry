¿Que es HTML?

Asi se ve un codigo basico de una web echa con HTML

¿Que es todo eso de "H1, P y H2"?

Lo que se ve en la imagen se conoce como etiquetas y se utilizan para darle "estilo" a la web

¿Que tipo de etiquetas existen?

Hay muchos tipos de etiquetas pero las principales son:

  • h, esta tiene la particularidad de funcionar con un sitema de gerarquia la cual nos permite definir encabezados organizadamente.
  • p, nos permite crear parrafos dentro de nuestra web.
  • form, nos permite crear un formulario.
  • input, nos sirve para definir un campo de entrada dentro de un formulario.
  • label, nos permite definir una etiqueta para un elemento.
  • button, nos permite crear un boton clickeable.
  • ul, las que nos permitio crear esta lista que si bien tiene un orden, nada lo define.
  • ol, al contrario de la anterior, esta nos permite crear listas ordenadas.
  • li, nos permite crear items dentro de las listas.
